# DeFi vs Banks

Compare decentralized finance protocols with traditional banks worldwide.

**[defivsbanks.com](https://defivsbanks.com)**

## What it does

Rankings and comparison of DeFi protocols alongside traditional banking institutions. Side-by-side data on TVL (total value locked) vs consolidated assets across:

- **DeFi protocols** (170+)
- **Centralized exchanges** (30+)
- **US banks** (200+)
- **Argentine banks** (70+)

Data sourced from DeFiLlama, Federal Reserve, and BCRA (Argentine central bank).

Inspired by [aavevsbanks.com](https://aavevsbanks.com). For educational purposes only.
